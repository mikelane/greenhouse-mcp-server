# Security Policy

## Reporting a Vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Report privately via [GitHub Security Advisories](https://github.com/mikelane/greenhouse-mcp-server/security/advisories/new) or email the maintainer directly.

Include: description, reproduction steps, affected versions, and any suggested fix.

| Milestone          | Target          |
|--------------------|-----------------|
| Acknowledgement    | 48 hours        |
| Initial assessment | 5 business days |
| Public disclosure  | After fix       |
